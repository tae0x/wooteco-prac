# 자동차 경주 게임

## 기능 목록

### 1. 경주 설정

- [x] 경주할 자동차 이름을 입력 받는다.
    - 이름은 쉼표를 기준으로 구분
- [x] 시도할 횟수를 입력받는다.

### 2. 경주 진행

- [x] 각 차수 별로 자동차들이 전진 또는 멈춘다.
    - 0에서 9 사이의 무작위 값을 구한다.
    - 무작위 값이 4 이상일 경우 전진
- [x] 차수별 실행 결과를 출력한다.

### 3. 우승자 선정

- [x] 가장 많이 전진한 자동차를 우승자로 선정한다.(공동우승 가능)
- [x] 최종 우승자를 출력(쉼표로 구분)

### 4. 예외 처리

- [x] 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시키고 종료
    - 자동차 이름이 5글자를 초과한 경우
    - 자동차 이름이 빈 값인 경우
    - 시도 횟수가 숫자가 아닌 경우
    - 시도 횟수가 0이하인 경우
