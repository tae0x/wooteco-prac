# 문자열 덧셈 계산기

## 기능 목록

### 입력

- [ ] 덧셈할 문자열을 입력 받기
- [ ] 입력 문구: 덧셈할 문자열을 입력해 주세요.

### 구분자 처리

- [ ] 쉼표(,) 구분자로 문자열 분리
- [ ] 콜론(:) 구분자로 문자열 분리
- [ ] 커스텀 구분자를 지원("//"과 "\n" 사이에 위치하는 문자를 구분자로 사용)

### 계산

- [ ] 구분자로 분리된 각 숫자의 합을 구함
- [ ] 빈 문자열이 입력되면 0을 반환

### 예외처리

- [ ] 음수가 입력되는 경우 IllegalArgumentException을 발생
- [ ] 숫자 이외의 값이 포함되는 경우 IllegalArgumentException을 발생
- [ ] 형식이 올바르지 않은 경우 IllegalArgumentException을 발생

### 출력

- [ ] 최종 연산 결과를 출력
- [ ] 출력 형식: 결과 : {숫자}
