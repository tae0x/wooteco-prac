# 문자열 덧셈 계산기

## 기능 목록

### 입력

- [x] 덧셈할 문자열을 입력 받기
- [x] 입력 문구: 덧셈할 문자열을 입력해 주세요.

### 구분자 처리

- [x] 쉼표(,) 구분자로 문자열 분리
- [x] 콜론(:) 구분자로 문자열 분리
- [x] 커스텀 구분자를 지원("//"과 "\n" 사이에 위치하는 문자를 구분자로 사용)

### 계산

- [x] 구분자로 분리된 각 숫자의 합을 구함
- [x] 빈 문자열이 입력되면 0을 반환

### 예외처리

- [x] 음수가 입력되는 경우 IllegalArgumentException을 발생
- [x] 숫자 이외의 값이 포함되는 경우 IllegalArgumentException을 발생
- [x] 형식이 올바르지 않은 경우 IllegalArgumentException을 발생

### 출력

- [x] 최종 연산 결과를 출력
- [x] 출력 형식: 결과 : {숫자}

### 리팩토링

- [x] 메서드 분리: 'calculate' 메서드 하나에 너무 많은 로직 존재(구분자 추출, 숫자 분리, 더하기)
- [x] 클래스 분리: 'Application' 클래스는 실행만 담당, 계산 로직은 별도 클래스로 분리
- [x] 상수 처리: 코드 내에 하드코딩된 문자열등을 상수로 추출
