## 기능 목록

### 입력 기능

-[ ] 투입 금액 입력 받기
    - 숫자 검증
    - 0 이상 검증

-[x] 보유 금액 입력 받기
    - 숫자 검증
    - 0 이상 검증

-[ ] 상품 정보 입력
    - 주어진 형식 적용

-[ ] 구매 상품명 입력
    - 속성값 검증

### 자판기 기능

-[x] 보유한 동전 랜덤 생성 (500,100,50,10)
-[x] 동전 보유 현황 출력
-[ ] 잔돈 반환
    - 동전으로만 받음(코인 값)
    - 잔돈 반환 조건: 남은 금액 < 상품들의 최소값 or 상품 수량
    - 반환할 수 있는 금액 전부 반환, 남은 금액 출력

### 상품 기능

-[ ] 상품 추가(이름, 가격, 수량)
    - 가격 100원 이상, 10원 단위
    - 각 속성값 검증

-[ ] 상품 구매(재고 감소)
    - 구매 가능 여부 확인
