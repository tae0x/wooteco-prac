## 기능 구현 목록

### 입력

- [ ] 자판기 보유 금액 입력
- [ ] 상품 정보 입력 ([이름,가격,수량];[이름,가격,수량])
- [ ] 투입 금액 입력
- [ ] 구매 상품명 입력

### 자판기

- [ ] 보유 금액으로 동전 랜덤 생성 (500, 100, 50, 10)
- [ ] 동전 보유 현황 출력
- [ ] 잔돈 반환 (최소 개수, 가능한 만큼만)

### 상품

- [ ] 상품 추가 (이름, 가격, 수량)
- [ ] 상품 구매 (재고 감소)
- [ ] 구매 가능 여부 확인

### 예외 처리

- [ ] 입력 유효성 검증
- [ ] 가격 100원 이상, 10원 단위
