# 로또 

## 기능 목록

### 입력
- [x] 구입 금액 입력 받기
- [x] 당첨 번호 입력 받기(쉼표로 구분)
- [x] 보너스 번호 입력 받기

### 로또 발행 
- [x] 구입 금액에 맞춰 로또 발행(1장당 1,000원)
- [x] 1~45 사이의 중복되지 않는 6개의 숫자를 무작위로 로또 번호 생성
- [ ] 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 판단
  - 1등: 6개 번호 일치 / 2,000,000,000원
  - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
  - 3등: 5개 번호 일치 / 1,500,000원
  - 4등: 4개 번호 일치 / 50,000원
  - 5등: 3개 번호 일치 / 5,000원
- [ ] 수익률 계산

### 출력
- [ ] 구매한 로또 개수와 번호 출력(오름차순)
- [ ] 당첨 통계 출력
- [ ] 수익률 출력(소수점 둘째 자리 반올림)
- [ ] 예외 상황 시 에러 메시지 출력([ERROR]로 시작)

### 예외 처리
- [ ] 구입 금액이 숫자가 아닌 경우 
- [ ] 구입 금액이 1000원 단위가 아닌 경우
- [ ] 당첨 번호가 1-45 범위를 벗어나는 경우
- [ ] 당첨 번호가 중복되는 경우
- [ ] 당첨 번호가 6개가 아닌 경우
- [ ] 보너스 번호가 당첨 번호와 중복되는 경우
