## 기능 목록

### 입력

- [x] 구매 금액 입력
    - [x] 숫자 검증
    - [x] 1,000원 단위 검증(양수 포함)

- [x] 당첨 번호 입력 받기
    - [x] 6개 숫자 검증
    - [x] 중복 검증
    - [x] 1~45 사이 값 검증

- [ ] 보너스 번호 입력 받기
    - [ ] 당첨 번호와 중복 검증
    - [ ] 숫자 검증
    - [ ] 1~45 사이 값 검증

### 로또 발행

- [x] 구입 금액만큼 로또 생성
- [x] 1~45 중 중복 없는 6개 번호 생성
- [x] 번호 오름차순 정렬
- [x] 로또 생성 번호 출력

### 당첨 계산

- [ ] 당첨 번호와 각 로또 번호 비교
- [ ] 일치 개수 계산
- [ ] 보너스 번호 일치 확인
- [ ] 등수 판정
- [ ] 등수 별 개수 집계

### 결과 출력

- [ ] 당첨 통계 출력
- [ ] 총 상금 계산
- [ ] 수익률 계산(소수점 둘째자리 반올림)
- [ ] 수익률 출력
