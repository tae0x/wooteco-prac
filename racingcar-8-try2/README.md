# 자동차 경주 게임

## 기능 목록

### 1. 입력

- [x] 경주할 자동차의 이름을 입력
    - 자동차의 이름은 쉼표로 구분
    - 이름의 5자를 초과할 경우 IllegalArgumentException 발생
- [x] 시도할 횟수를 입력
    - 숫자가 아닌 경우 IllegalArgumentException 발생
    - 0 이하의 숫자일 경우 IllegalArgumentException 발생

### 2. 게임 진행

- [x] 각 자동차 마다 0에서 9 사이의 무작위 값을 구함
- [x] 무작위 값이 4 이상인 경우 자동차를 전진
- [x] 매 회차마다 실행 결과를 출력

### 3. 결과 출력

- [x] 전진 거리가 가장 먼 자동차를 우승자로 선정
- [x] 우승자가 여러 명일 경우 쉼표를 이용하여 구분
- [x] 최종 우승자를 출력
